1.Apakah program berjalan lancar? Ya, program berjalan lancar tanpa error dan tanpa perlu mengubah kode looping. Kesimpulan: Polymorphism memungkinkan programmer menambahkan karakter baru dengan mudah—cukup buat class Healer dengan method serang(), masukkan ke list pasukan, dan semua langsung bekerja otomatis tanpa mengubah kode yang sudah ada, sehingga update game di masa depan menjadi lebih cepat, aman, dan tidak berisiko merusak fitur lama.

2.Apa yang terjadi? Program error: AttributeError: 'Archer' object has no attribute 'serang'. Mengapa harus sama? Dalam Polymorphism, Python memanggil method berdasarkan nama yang konsisten (serang()) untuk semua objek—jika nama method di Archer diubah menjadi tembak_panah(), Python tidak menemukan method serang() saat looping sehingga error, karena Polymorphism membutuhkan interface yang seragam agar satu perintah (pahlawan.serang()) bisa bekerja untuk berbagai tipe objek berbeda.
